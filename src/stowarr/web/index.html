Guide

How Stowarr pages, buttons, and safety boundaries work

Sync

Compares qBittorrent hashes, categories, save paths, and authoritative pools with Radarr or Sonarr history and library roots.

Run audit
Refreshes the selected Radarr or Sonarr comparison.
Reconcile
Opens a repair plan for the selected torrent hash.

Overview

Summarizes operations, completed and blocked work, configured pools, routing, and recent activity.

View all
Opens the complete operation history.
Inspect
Builds a current Reconcile plan for that torrent.

Reconcile

Repairs an inconsistency using qBittorrent as the source of truth. It verifies content before creating hardlinks, updating *Arr, or removing a stale derivative.

Analyze
Builds a read-only plan from an info hash.
Verify content hashes
Hashes torrent, old library, new library, and selected sidecars.
Reconcile
Requests confirmation and executes the verified plan.

Move

Moves a torrent between pools through qBittorrent, rebuilds the Radarr/Sonarr library, processes every additional file, and removes the empty old folder last.

Refresh torrents
Reloads qBittorrent and clears the current selection.
Column buttons
Show or hide Radarr, Sonarr, and unmanaged route columns.
Build move plan
Inventories torrent, *Arr, download, and library files without changing them.
Move all / Delete all
Sets the proposed action for every additional file.
Review & move
Shows a final summary, issues a bound confirmation, and starts the transaction.

History

Lists Move and Reconcile attempts with their final or current transaction state.

Inspect
Rebuilds a current plan; it does not replay the historical operation.

Settings

Manages service credentials and runtime execution mode, discovers routing, and displays pool and Docker deployment configuration.

Open connection setup
Tests and saves one or more configured services without blocking the WebUI.
Save execution mode
Enables or disables confirmed writes after checking mount access.
Discover routes
Reads qBittorrent destinations first, then matches *Arr clients, categories, tags, and roots.

Detected Radarr, Sonarr, and qBittorrent route configuration
